# ediop3Cam
Unconfigured cctv cameras finder.   Example link "http://67.53.204.42:8001"



## installation
```
git clone https://github.com/ediop3SquadALT/ediop3Cam
cd ediop3Cam
bash ediop3Cam.sh
```
